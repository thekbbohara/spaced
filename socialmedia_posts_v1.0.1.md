### X thread (the build-in-public version)

```
1/ Last week I pre-released Spaced — a tiny, offline flashcard app built on active recall + spaced repetition.

Then I did the one thing that actually finds bugs: I used it every day for a week. Here's everything that week turned into 🧵

2/ 🗂 Folders. After ~20 topics it was a mess, so I added a folder tree: class → subject → unit → chapter, nested to any depth, every level optional. Plus per-folder stats and "study a whole folder."

3/ 🔀 "Study all due." Opening each deck one by one got old fast. Now one button mixes every card that's due across all decks — Again / Hard / Good, skip, and undo, all in one flow.

4/ 🧩 Cloze + images. I kept wanting fill-in-the-blank cards and diagrams, so: cloze deletion ({{c1::...}}) and images on the front/back of any card.

5/ 📥 Import/export + 🔍 search. I had decks living in Anki, so I added Anki TSV / CSV import and CSV export. And full-text search across every topic and card, because scrolling to find one card is painful.

6/ A few more from that week: 5-minute recall sessions, a full offline activity log, heads-up reminders with a daily digest, and an accessibility pass (screen-reader labels + font scaling).

7/ Still 100% offline. No account, no ads, no tracking. Open source. It's an early beta and I'd love your feedback.
APK + code: https://github.com/thekbbohara/spaced
```
